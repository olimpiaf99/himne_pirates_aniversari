# Himne 50-75 Aniversari de la Filà Pirates d'Oliva
Este repositorio genera una web en la que se podrá ver la letra del himno en valenciano, español e inglés.
El estilo está basado en la web de lyrics Genius.com, en la que se subrayan palabras o frases de la canción de forma que tu puedes hacer click y te aparece una breve desprición de lo que significa esa parte de la letra.
